Technical Report: Deepfake Detection Using Multi-Architecture Deep Learning Models
Acknowledgements
We acknowledge the contributions of the deep learning research community, particularly the authors of EfficientNet, FasterViT, and EfficientFormerV2 architectures. Special thanks to the PyTorch team and the creators of the grad-CAM visualization technique.

Abstract
This report presents a comprehensive analysis of a deepfake detection system that leverages multiple deep learning architectures to achieve robust face authentication. The system combines Convolutional Neural Networks (CNNs) and Vision Transformers (ViTs) through three distinct models: EfficientNet-B3, FasterViT-2-224, and EfficientFormerV2-S1. The implementation includes real-time inference capabilities and attention visualization through Gradient-weighted Class Activation Mapping (Grad-CAM). Our results demonstrate complementary strengths across architectures, with a user-friendly interface for practical deployment.

1. INTRODUCTION
1.1 Overview
The project implements a multi-model deepfake detection system that analyzes facial images to determine authenticity. It employs three state-of-the-art deep learning architectures and provides visual explanations for model decisions through attention mapping.

1.2 Background
With the rise of sophisticated deepfake technologies, reliable detection methods have become crucial for digital media authenticity. Deep learning approaches, particularly hybrid architectures combining CNNs and transformers, have shown promising results in this domain.

1.3 Motivation
The proliferation of deepfake content poses significant societal challenges, from misinformation to privacy concerns. This necessitates robust detection systems that can operate reliably across different types of synthetic media.

1.4 Purpose and Objectives
The primary objectives include:

Development of a multi-architecture detection system
Integration of complementary model strengths
Provision of visual explanations for model decisions
Creation of an accessible interface for non-technical users
